APP_KEY ungültig

Der Schlüssel in der Datei .env (APP_KEY) ist leer oder hat eine falsche Länge. Laravel benötigt einen gültigen Schlüssel im Format base64:... (für AES-256-CBC: 32 Bytes).

So beheben Sie das:

  1. Per SSH im Projektordner: php artisan key:generate ausführen. Das setzt APP_KEY in .env auf einen gültigen Wert.
  2. Oder in .env die Zeile APP_KEY= leer lassen und unten „Install-Seite erneut aufrufen“ klicken – der Installer versucht dann, einen Schlüssel zu erzeugen (falls .env beschreibbar ist).
