ZerroCMS – .env manuell anlegen

Die Datei .env konnte nicht automatisch erstellt werden (z. B. fehlende Schreibrechte im Projektordner auf dem Server).

So beheben Sie das:

  1. Im Projektordner (dort, wo artisan liegt) eine Datei .env anlegen.
  2. Entweder .env.example dorthin kopieren und als .env speichern.
  3. Oder den folgenden Mindestinhalt in die neue Datei .env eintragen, speichern und Werte (Datenbank, APP_URL) anpassen:
APP_NAME=ZerroCMS
APP_ENV=production
APP_KEY=
APP_DEBUG=false
APP_URL=https://meinedomain.de

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=zerrocms
DB_USERNAME=root
DB_PASSWORD=

SESSION_DRIVER=file
CACHE_STORE=file
QUEUE_CONNECTION=sync

CLAN_NAME=ZerroCMS
CLAN_THEME=default

Anschließend entweder per SSH php artisan key:generate ausführen, oder den Link unten nutzen – der Installer erzeugt den Schlüssel beim nächsten Aufruf.
