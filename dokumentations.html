Was ist ZerroCMS?

Clan CMS für Gaming-Communities
ZerroCMS ist ein vollständiges Content-Management-System speziell für Clans und Gaming-Communities. Es läuft auf Laravel (PHP) mit Filament als Admin-Panel und bietet alles, was ein Clan braucht: News, Forum, Galerie, Kalender, Bewerbungen, Crafting/Aufträge, Marketplace, Wiki, Clan-Bank, Teams, Leaderboards und vieles mehr. Für Shared Hosting geeignet – kein Redis nötig.
Technologie
Laravel 12 · Filament 5 · Livewire · Tailwind CSS · Blade

Installation

Variante A: Webbrowser-Installer (empfohlen)

  1. Abhängigkeiten installieren: composer install
  2. Im Browser /install aufrufen
  3. Dem Assistenten folgen: Systemprüfung → Datenbank → Migrationen → Admin-User anlegen
  4. Speicher-Link erstellen: php artisan storage:link
  5. Für Produktion: npm ci && npm run build

Variante B: Manuell

cp .env.example .env
php artisan key:generate
# .env: DB_HOST, DB_DATABASE, DB_USERNAME, DB_PASSWORD eintragen
php artisan migrate
php artisan db:seed
php artisan storage:link

Admin-Panel

Zugang unter /admin. Login mit Admin-User (Rolle super-admin oder Berechtigung access_admin).

Bereiche im Admin
Ränge, Mitglieder, Bewerbungen, Termine (Events), News, Seiten, Forum, Galerie, Crafting (Craftable Items, Item Requests), Clan-Dokumente, Marketplace (Kategorien, Anzeigen), Wiki, Jobs, Downloads, Partner, Umfragen, Clan-Teams, Clan-Bank, Clan-Regeln, Leaderboard, Clan-Dokumente, Clan-Feedback, Clan-Ankündigungen, Clan-Achievements, Widget-Instanzen, Theme-Editor, Einstellungen (Site, Medien, Wartung, Bewerbungen, Frontend), Menüs, Rechteverwaltung, Module, Plugins.
Discord Quick-Befehle
Unter Discord → Quick-Befehle kannst du Slash-Commands anlegen (z. B. Name spitzhacke, Antworttext „Wer braucht eine Spitzhacke? …“). Diese werden im Discord-Bot als /befehl run name:spitzhacke nutzbar.

Discord-Bot

Slash-Commands & Integration
Der Discord-Bot (Node.js, Ordner discord-bot/) ergänzt ZerroCMS um Slash-Commands in Discord. Shop-Angebote und Events werden automatisch von Laravel in Discord-Channels gepostet (Webhooks). Der Bot wird nur für Slash-Commands und Spieler-Abfrage benötigt.
Slash-Commands
/befehl run name:<name> – Führt einen auf der Website angelegten Quick-Befehl aus (z. B. name:spitzhacke).
/befehl list – Listet alle verfügbaren Quick-Befehle (nur für dich sichtbar).
/spieler – Zeigt deinen verknüpften Website-User (wenn Discord mit User verknüpft).
Automatische Posts (ohne Bot)
Neue Marketplace-Anzeige → Post im Shop-Channel (Webhook DISCORD_SHOP_WEBHOOK_URL).
Neues/geändertes Event → Post im Events-Channel (Webhook DISCORD_EVENTS_WEBHOOK_URL).
Bot starten
cd discord-bot → cp .env.example .env → .env anpassen (DISCORD_BOT_TOKEN, LARAVEL_API_URL, DISCORD_BOT_API_KEY) → npm install && npm start

Anforderungen

  • PHP 8.2+
  • Composer
  • MySQL/MariaDB
  • PHP-Extensions: pdo, pdo_mysql, mbstring, openssl, tokenizer, xml, ctype, json, bcmath, fileinfo
  • Optional: Node.js/npm für Frontend-Build (sonst CDN-Fallback)

Hosting & Deployment

  • Document Root auf public/ zeigen
  • Queue: QUEUE_CONNECTION=sync falls kein Redis/Cron
  • Produktion: npm run build, php artisan config:cache
  • Status-Check: GET /status liefert JSON mit ok, database, installed
